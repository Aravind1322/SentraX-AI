# 🛡️ SentraX AI

SentraX AI is an AI-powered cybersecurity platform designed to detect and analyze digital threats through intelligent scanning and enterprise-focused investigation workflows.

---

## 🚀 Features

### 🌐 URL Scanner
- Detects suspicious URLs.
- Identifies phishing indicators.
- Provides instant threat feedback.

### 📩 SMS Scam Detector
- Analyzes SMS messages for scam patterns.
- Detects spam and suspicious keywords.

### ⚡ Quick Scan
Universal intake portal for:
- CSV files
- TXT files
- PDF files

Automatically detects the dataset type and routes it to the appropriate analysis engine.

Supported datasets:
- Fraud Transactions
- URL Intelligence
- SMS Analysis
- Employee Monitoring

Downloads:
- CSV Reports
- TXT Reports
- PDF Reports
